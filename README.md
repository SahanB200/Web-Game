# 🏃 Real-Time Running Game

This is a fun real-time browser-based running game made with pure **HTML**, **CSS**, and **JavaScript** — no external libraries required!

## 🎮 Gameplay Overview

- Press **Enter** to start running.
- Press **Space** to jump over flames.
- If you hit a flame while not jumping — **Game Over**.
- Reach a score of **5000** to **Win**!
